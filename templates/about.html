Sobre o Graphy

Graphy é uma plataforma web desenvolvida para tornar a computação gráfica acessível a todos. Com ela, você pode aplicar filtros clássicos de processamento de imagens diretamente do navegador, sem instalar nada e de forma totalmente gratuita.

Por que Graphy?

O projeto nasceu como um trabalho acadêmico para a disciplina de Computação Gráfica, ministrada pelo docente André Luiz Batista Franca. Nosso objetivo foi criar uma ferramenta prática, intuitiva e educativa, que demonstra na prática conceitos fundamentais de processamento digital de imagens.

Funcionalidades

  • Upload de múltiplas imagens (JPG, PNG)
  • Aplicação de filtros como Negativo, Mediana, Gaussiano, Escala de Cinza e Preto & Branco
  • Visualização instantânea e download das imagens processadas
  • Interface responsiva e fácil de usar
  • Privacidade: suas imagens não ficam salvas no servidor

Como funciona?

O Graphy utiliza Python com Flask no backend e a biblioteca Pillow para manipulação de imagens. O frontend é construído com HTML, CSS (Bootstrap) e JavaScript, proporcionando uma experiência moderna e fluida.
